# Docs

This directory contains documentation for `coding-agent-safety-gate`. Read in this order if you are new to the project.

## Reading order

| Order | File | What it covers |
|---|---|---|
| 1 | `safety-boundaries.md` | The default safety rules: secret-safe, path-safe, git-safe, env-safe, execution-safe, evidence-based acceptance, and why this is one layer, not the whole safety system |
| 2 | `human-in-the-loop.md` | Division of responsibility between humans and AI agents, multi-agent collaboration flow, required human decision points |
## After reading

- Return to the root `README.md` for the overview.
- Open `QUICKSTART.md` to start using the project.
- Copy templates from `templates/` into your own project.
